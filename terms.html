Persyaratan Layanan

Selamat datang di Sistem Analisis Sentimen TikTok. Dengan menggunakan layanan ini, Anda menyetujui ketentuan berikut:

  • Layanan ini dikembangkan untuk keperluan penelitian akademik dan tidak digunakan untuk tujuan komersial.
  • Data yang dianalisis bersumber dari konten publik dan tidak menyimpan data pengguna secara permanen.
  • Sistem menggunakan analisis otomatis berbasis Natural Language Processing dan tidak menjamin hasil yang sepenuhnya akurat.
  • Penggunaan sistem untuk menyebarkan informasi palsu, ujaran kebencian, atau manipulasi data dilarang keras.
  • Pengembang berhak mengubah atau menghentikan layanan ini kapan saja tanpa pemberitahuan terlebih dahulu.

Untuk pertanyaan lebih lanjut, silakan hubungi pengembang melalui halaman GitHub ini.
